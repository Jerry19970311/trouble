1，更换上传的远程地址

git 删除远程地址：git remote rm origin
git 更换远程地址：git remote add origin http://10.00.3.260:10080/xxxxxx/sxxxy.git/
把当前分支与远程分支进行关联(branchname要改成你的当前分支名称)：git push --set-upstream origin branchname
